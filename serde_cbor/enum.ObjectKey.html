pub enum ObjectKey {
    Integer(i64),
    Bytes(Vec<u8>),
    String(String),
    Bool(bool),
    Null,
}

A simplified CBOR value containing only types useful for keys.

Variants

Integer(i64)

An integer.

Bytes(Vec<u8>)

A byte string.

String(String)

An UTF-8 string.

Bool(bool)

A boolean value.

Null

No value.
